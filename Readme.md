Reader
---
此项目为测验练手用，是一个简易的分级列表阅读程序。

Demo下载
---
https://github.com/kelv1nq1an/Reader/releases

进度
---
v1.0完成所有要求，进入测试阶段。
v1.1添加高亮文字点击提示。
v1.2文章界面更改背景，slidebar添加常驻文字提示。

要求
---
1. 打开后看到文章列表
2. 点击列表某项，打开文章
3. 文章界面有一个按钮，点击则在文章中高亮在单词列表中出现的单词
4. 文章界面有一个slide-bar，从0到6（对应单词表中每个单词信息的level)，可以拖动过滤高亮的单词。
